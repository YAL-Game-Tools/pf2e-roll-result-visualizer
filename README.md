# YAL's PF2e roll result visualizer

For the burning Pathfinder 2nd edition questions like "does it make sense to grapple this enemy", "how much does a +1 status bonus help", or "should I attack with MAP?".

See [AoN](https://2e.aonprd.com/Rules.aspx?ID=2278) for full rules.

A tool by [YellowAfterlife](https://yal.cc/), written in [Haxe](https://haxe.org/).

## Building

Install Haxe (see above);
```
haxe build.hxml
```
If all is well, a new `script.js` should manifest in the `bin/` directory.
